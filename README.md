#Dotfiles

## Firefox

### Theme

1. Create a new profile using:

```sh
firefox -no-remote -ProfileManager
```

2. Set it to default, and note it's directory name

3. Follow the instructions at: https://github.com/Zortax/Vimix-Midnight-Firefox
